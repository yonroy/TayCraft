RNN một bước — lời giải & kiểm tra

1

Wx·xₜ

= (1·2, −1·2) = ( 2, −2 )
2

Wh·hₜ₋₁

hàng 0: 1·1 + 0·(−1) = 1
hàng 1: −1·1 + 1·(−1) = −2
3

z = (1)+(2)+b

z₀ = 2 + 1 + 0 = 3
z₁ = −2 + (−2) + 1 = −3
4

hₜ = tanh(z)

hₜ = ( tanh(3), tanh(−3) )
hₜ = ( 1.00, −1.00 )
Rút ra. hₜ mới = ( 1.00, −1.00 ) giờ sẽ làm hₜ₋₁ cho bước thời gian kế tiếp, dùng chính bộ Wx, Wh, b này — đó là "hồi quy". Vì mọi bước nhân lặp cùng Wh, gradient dễ tiêu biến hoặc bùng nổ với chuỗi dài → đó là động lực ra đời LSTM (Bài 22), thêm cổng để giữ ký ức bền hơn.
