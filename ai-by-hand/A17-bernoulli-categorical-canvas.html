Bernoulli & Categorical

— Phân phối rời rạc P(0)+P(1)=1 · Σpᵢ=1
Vì sao quan trọng. Hai phân phối này là đầu ra của mô hình phân loại. Bernoulli (p): phân loại nhị phân — "thư rác hay không?" (output: sigmoid → p). Categorical (p₁,…,pₖ): phân loại K lớp — "chữ số 0–9?" (output: softmax → p₁,…,p₁₀). Cross-entropy loss đo khoảng cách từ phân phối dự đoán đến phân phối nhãn thật.
Bernoulli(p): P(X=1)=p, P(X=0)=1−p  |  Categorical: Σpᵢ=1, pᵢ≥0
💡 Trực giác Bernoulli = đồng xu lệch. Categorical = con súc xắc không đều K mặt. Sigmoid biến z → p ∈ (0,1) (Bernoulli). Softmax biến z₁…zₖ → (p₁…pₖ) (Categorical).

Cross-entropy: −log(pₐᵣgₘₐₓ) nếu nhãn thật là argmax. Nhỏ khi mô hình tự tin đúng.
BERNOULLI(p) · p = 0.3
P(X=0) = 0.7    P(X=1) = 0.3
E[X] = p = 0.3    Var(X) = p(1−p) = 0.21
CATEGORICAL · 4 lớp
Tổng p = 1.00    argmax = lớp 1
✎ Tự kiểm tra — đáp án
  1. Bernoulli(0.7): E[X] = 0.7; Var(X) = 0.7×0.3 = 0.21
Rút ra. Phân phối Categorical = đầu ra của bất kỳ mô hình phân loại nào. Cross-entropy loss = −ln(pᵧ) với pᵧ là xác suất gán cho nhãn đúng. Khi mô hình chắc chắn đúng (pᵧ → 1): loss → 0. Khi sai (pᵧ → 0): loss → ∞. Bài B3 (Logistic Regression) dùng Bernoulli + cross-entropy; B1–B11 dùng Categorical.
