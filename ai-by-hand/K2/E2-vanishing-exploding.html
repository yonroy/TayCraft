Vanishing / Exploding gradient — teo / nổ

Họ tên: Ngày: Thời gian: ~12 phút
Vì sao quan trọng. Gradient qua mạng sâu là tích nhiều thừa số. Nếu mỗi thừa số < 1 → teo về 0 (vanishing): lớp đầu gần như không học; nếu > 1 → nổ (exploding): cập nhật loạn, loss thành NaN. Hiểu nó dẫn tới mọi cách chữa.
∂L/∂(lớp đầu) = ∏ (đạo hàm cục bộ mỗi lớp)    rᴸ → 0 nếu r<1 · → ∞ nếu r>1
💡 Trực giác Gradient truyền ngược như tiếng vọng qua L căn phòng: mỗi phòng nhân âm lượng với r. r=0.5 qua 10 phòng còn ~0.001 (tắt ngấm); r=1.5 thành ~58 (nổ tai). Chỉ r≈1 mới giữ tín hiệu ổn định.

Gradient theo độ sâu (3 giá trị r)

teo / ổn định / nổ
Đường xám nét đứt = r<1 → teo về 0 (vanishing). Lam = r=1 → ổn định. Cam = r>1 → nổ (exploding). Cùng số lớp, chỉ khác r mà số phận gradient khác hẳn.
✎ Tự kiểm tra — đáp án
  1. Sigmoid sâu vanishing? → σ' ≤ 0.25 < 1, nhân dồn qua nhiều lớp → teo về 0.
  2. Gradient ổn định? → Hệ số khuếch đại mỗi lớp ≈ 1.
Rút ra. Gradient = tích thừa số; r<1 teo, r>1 nổ, cần r≈1. Bài tiếp (E3): một cách kiểm soát độ lớn trọng số — L2 regularization (weight decay).
