Seq2seq

— Encoder → Decoder nén chuỗi → context c → giải mã
Vì sao quan trọng. Dịch máy, tóm tắt… cần biến một chuỗi thành chuỗi khác. Seq2seq dùng encoder đọc hết câu vào, nén thành một vectơ ngữ cảnh c (trạng thái ẩn cuối), rồi decoder sinh từng token đầu ra từ c. Đơn giản nhưng c là nút thắt cổ chai — động lực ra đời attention.
encoder: hₜ = hₜ₋₁ + xₜ → c = h cuối  ·  decoder: sₜ = ½·sₜ₋₁, s₀ = c (ở đây w_enc=1, w_dec=½)
💡 Trực giác Encoder "đọc xong rồi gói gọn" toàn bộ câu vào một con số/vectơ c; decoder "mở gói" để viết câu ra. Mọi thông tin đầu vào phải lọt qua c — câu càng dài, c càng quá tải (nút thắt).
Lam = encoder nén dần · c = nút thắt (vectơ ngữ cảnh) · cam = decoder bung ra.
