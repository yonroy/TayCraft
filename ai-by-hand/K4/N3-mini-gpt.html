Họ tên: Ngày: Thời gian: ~16 phút
Vì sao quan trọng. LLM thu nhỏ làm tay được từ id token tới token sinh ra. Thấy rõ cách mọi mảnh attention ghép thành "máy đoán chữ kế". Hiểu nó là nắm xương sống của ChatGPT.
token → Embed+PE → [Self-Attn (causal) → FFN] → logits → softmax → argmax = token kế
💡 Trực giác Hai token đã có → mô hình trộn ngữ cảnh (token cuối nhìn lại token trước qua attention), xử lý (FFN), rồi chấm điểm mọi token từ điển → chọn token điểm cao nhất làm chữ kế. Lặp = viết câu.
